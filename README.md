# React Fundamentals

This repository contains **simple React fundamentals lessons** to help understand the basics of React in a clean and organized way.

---

## ğŸš€ Getting Started

Follow these steps to run the project locally.

1. **Clone the repository**
   ```bash
    git clone https://github.com/JekenMaharjan/react-fundamentals.git
    ```
   
2. **Navigate to the project directory**
   ```bash
    cd client
    ```

3. **Install dependencies**
   ```bash
    npm install
    ``` 
    or
    ```bash
    yarn install
    ```

4. **Start the development server**
   ```bash
    npm run dev
    ```
    or
    ```bash
    yarn dev
    ```

5. **Open in browser**
   ```bash
    http://localhost:5173
    ```
